👩‍🏫 How to add a screenshot for an app

The preview box shows a screenshot image if you supply one. Just:

  • Take a screenshot of your app (any screenshot tool is fine)
  • Name it exactly the same as the HTML file but with .png — e.g. pizza-fractions.png
  • Upload it to the screenshots/ folder in your repo
💡 No screenshot yet? No problem — a colourful placeholder shows instead, and the Launch button still works perfectly!
